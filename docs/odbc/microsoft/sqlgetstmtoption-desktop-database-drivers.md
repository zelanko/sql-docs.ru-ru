# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (драйверы для баз данных на настольном компьютере)
Закладки, возвращаемые *параметром foption* SQL_GETBOOKMARK, являются допустимыми только тогда, когда запрос открыт и становится недействительным при повторной выдаче запроса. Постоянные закладки не поддерживаются.
