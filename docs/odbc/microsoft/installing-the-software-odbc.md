# <a name="installing-the-software-odbc"></a>Установка программного обеспечения (ODBC)
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Драйвер ODBC для Oracle — один из компонентов доступа к данным. Она прилагается к других компонентов ODBC, такие как источники данных ODBC и уже должны быть установлены. Драйвер также можно найти в разделе «Драйверы и другие файлы для загрузки» на сайте интерактивные службы поддержки продуктов Microsoft [www.microsoft.com](https://www.microsoft.com).  
  
 Сетевое программное обеспечение должно устанавливаться в соответствии с свою собственную документацию. Драйвер ODBC для Oracle требует без установки специальных предосторожностей до тех пор, пока поддерживается сетевое программное обеспечение.  
  
 Программное обеспечение Oracle должно устанавливаться в соответствии с свою собственную документацию. Драйвер ODBC для Oracle обычно особенностей нет специальной установки до тех пор, пока драйвер поддерживает версию. Тем не менее чтобы сохранить совместимость продуктов, необходимо установите драйвер ODBC для Oracle последней проверить наличие последней версии драйвера. Oracle поддерживает открытый FTP-сайта, где он учитывает, помимо прочего, исправления для серверных продуктов Oracle и клиентский компонент, который поставляется вместе с серверными продуктами. Эти обновления необходимы для правильной работы нескольких продуктов и технологий Microsoft. Дополнительные сведения об этом узле см. в разделе [исправления программного обеспечения Oracle](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Установка программного обеспечения Oracle через MDAC/Windows DAC могут быть перезаписаны текущие версии MDAC. При возникновении проблем с помощью компонентов ODBC, переустановите компоненты MDAC.
