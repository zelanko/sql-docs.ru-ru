# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Создание и удаление представлений и процедур с использованием ExtendedAnsiSQL
При включении флага ExtendedAnsiSQL Jet 4.0 поддерживает возможность создания и удаления представлений и процедур через SQL. Эти представления и процедуры нельзя увидеть с помощью доступа.
