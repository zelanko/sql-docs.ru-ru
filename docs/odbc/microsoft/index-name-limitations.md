# <a name="index-name-limitations"></a>Ограничения имени индекса
При использовании драйвера Paradox первичный индекс должен иметь имя, совпадающее с именем таблицы, на которой он определен. Другие индексы уникальный или неуникальный должен иметь имя, совпадающее с именем таблицы, на которой они определены.
