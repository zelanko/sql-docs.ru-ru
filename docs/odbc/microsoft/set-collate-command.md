# <a name="set-collate-command"></a>Команда SET COLLATE
Указывает порядок сортировки для символьных полей в последующих индексирования и операций сортировки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
SET COLLATE TO cSequenceName  
```  
  
## <a name="arguments"></a>Аргументы  
 *cSequenceName*  
 Указывает порядок сортировки. В следующей таблице описаны параметры последовательности доступных параметров сортировки.  
  
|Параметры|Язык|  
|-------------|--------------|  
|НИДЕРЛАНДСКИЙ|Нидерландский|  
|GENERAL|Английский, французский, немецкий, современных испанский, португальский и других европейских языках|  
|НЕМЕЦКИЙ|Порядок немецкой телефонной книги (DIN)|  
|ИСЛАНДИЯ|Исландский|  
|МАШИНЫ|Компьютер (сортировки по умолчанию для более ранних версий FoxPro)|  
|NORDAN|Норвежский, датский|  
|ИСПАНСКИЙ|Испанский (традиционный)|  
|SWEFIN|Шведский, финский|  
|UNIQWT|Уникальный вес|  
  
> [!NOTE]  
>  При указании параметра ИСПАНСКИЙ *ch* представляет одну букву, которая сортирует между *c* и *d*, и *ll* сортирует между  *l* и *m*.  
  
 При использовании параметра сортировки последовательности как строка буквенный символ, не забудьте заключить в кавычки параметр:  
  
```  
SET COLLATE TO "SWEFIN"  
```  
  
 КОМПЬЮТЕР является параметром по умолчанию параметры сортировки последовательности и является знакомы пользователям Xbase последовательности. Символы упорядочены, как они указаны в текущей кодовой странице.  
  
 Общие может быть предпочтительнее для пользователей США и Западная Европа. Символы упорядочены, как они указаны в текущей кодовой странице. В версиях FoxPro предшествующих 2.5, индексов может быть создана с помощью **верхний**() или **НИЖНЕЙ**функции () для преобразования символьных полей в согласованное регистр. В более поздней версии, чем 2.5 версии FoxPro, вместо этого можно указать параметр последовательности Общие параметры сортировки и опустить **верхний**преобразования ().  
  
 Если указать параметр последовательность параметров сортировки, отличный от компьютера, и если вы создадите файл .idx compact .idx создается всегда.  
  
 Используйте SET("COLLATE") для возврата текущей последовательности параметров сортировки.  
  
 Можно указать порядок сортировки для источника данных с помощью [ODBC Visual FoxPro диалоговое окно настройки](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md) или с помощью ключевого слова Collate в строке подключения с [SQLDriverConnect](../../odbc/microsoft/sqldriverconnect-visual-foxpro-odbc-driver.md). Этот подход аналогичен выдачи следующей команды:  
  
```  
SET COLLATE TO cSequenceName  
```  
  
## <a name="remarks"></a>Примечания  
 ЗАДАЙТЕ COLLATE позволяет порядке таблиц, содержащих символы с диакритическими знаками для любой из поддерживаемых языков. Изменение настроек ЗАДАТЬ COLLATE не влияет на порядок сортировки, ранее открывавшихся индексов. Visual FoxPro автоматически сохраняет существующие индексы, предоставляя гибкие возможности создавать много различных типов индексов, даже для одного поля.  
  
 Например если индекс создается с ЗАДАТЬ COLLATE присвоено общие и более поздней версии параметр ЗАДАТЬ COLLATE изменяется на испанский язык, индекс сохраняет общие параметры сортировки.
