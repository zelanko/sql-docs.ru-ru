# <a name="interoperability-of-sql-statements"></a>Совместимость инструкций SQL
Как в остальной части приложения инструкции SQL могут быть с возможностью взаимодействия или СУБД. Как в остальной части приложения, необходимо осторожно Выбор инструкций SQL, как с возможностью взаимодействия зависит от типа приложения. Пользовательские приложения обычно не использовать с возможностью взаимодействия инструкций SQL, поскольку обычно они предназначены для использования возможности одной или возможно, два СУБД. Универсальные приложения используйте с возможностью взаимодействия инструкции SQL, так как они предназначены для работы с различными СУБД. И вертикальные приложения обычно попадают где-нибудь в диапазоне от ресурсоемких уровень функциональности, но в противном случае с помощью с возможностью взаимодействия инструкций SQL.
