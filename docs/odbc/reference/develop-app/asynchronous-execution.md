1.  [Асинхронное исполнение (Метод опроса)](../../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)  
  
2.  [Асинхронное исполнение (метод уведомления)](../../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)
