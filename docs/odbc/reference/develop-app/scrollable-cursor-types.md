# <a name="scrollable-cursor-types"></a>Типы прокручиваемых курсоров
Четыре типа Прокручиваемые курсоры статические, динамические, управляемые набором ключей и смешанного. Статические курсоры обнаружения незначительных изменений, но являются относительно недорогим в реализации. Динамические курсоры обнаруживают все изменения, но трудно реализовать. Курсоры, управляемые набором ключей и смешанной лежать, обнаруживая большинство изменений, но с меньшими затратами, чем динамические курсоры.  
  
 Следующие термины используются для определения характеристик каждого типа Прокручиваемый курсор:  
  
-   **Собственные обновления, удаления и вставки.** Обновления, удаления и вставки, осуществленные через курсор, либо с помощью вызова **SQLBulkOperations** или **SQLSetPos** или с позиционированные инструкция update или delete.  
  
-   **Другие обновления, удаления и вставки.** Обновления, удаления и вставки, не выполненные курсор, числе и других операций в той же транзакции, с другими транзакциями и внесенными другими приложениями.  
  
-   **Членство.** Набор строк в результирующем наборе.  
  
-   **Порядок.** Порядок, в котором строки возвращаются курсором.  
  
-   **значения.** Значения в каждой строке в результирующем наборе.  
  
 Сведения о том, как обновлять, удалять и вставлять данные, см. в разделе [Обзор обновления данных](../../../odbc/reference/develop-app/updating-data-overview.md).
