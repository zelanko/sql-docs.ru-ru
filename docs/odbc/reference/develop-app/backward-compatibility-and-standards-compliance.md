# <a name="backward-compatibility-and-standards-compliance"></a>Обратная совместимость и соответствие стандартам
Обратная совместимость — это способность новых компонентов ODBC для работы с старые компоненты ODBC. В следующих разделах рассматривается, как эти компоненты были затронуты изменениями в ODBC 3 *.x*. Сведения, содержащиеся в них, в основном устраняет записи ODBC 3 *.x* приложения и как назад обрабатываются проблемы совместимости с драйверами ODBC. Дополнительные инструкции о том, как обратной совместимости неполадках записи ODBC 3 *.x* драйвера, см. в разделе [приложение G: драйвер рекомендации для обеспечения обратной совместимости](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md).
