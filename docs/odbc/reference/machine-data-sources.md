# <a name="machine-data-sources"></a>Машинные источники данных
*Машинные источники данных* хранятся в системе, где пользовательское имя. Связанный с именем источника данных необходим все сведения диспетчер драйверов и драйверов для подключения к источнику данных. Для источника данных Xbase это может быть имя драйвера Xbase, полный путь к каталогу, содержащему файлов Xbase и некоторые параметры, которые задают драйвер для использования этих файлов, например в однопользовательском режиме, или только для чтения. Для источника данных Oracle, это может быть имя драйвера Oracle, сервера, где находится СУБД Oracle SQL * Net строку подключения, которое идентифицирует SQL\*Net драйвер и идентификатор системы базы данных на сервере.
