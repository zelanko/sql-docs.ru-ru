# <a name="configuration-components"></a>Компоненты конфигурации
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включена в операционную систему Windows. ODBC следует только явным образом установить в более ранних версиях Windows.  
  
 Источники данных настроены с помощью установщика библиотеки DLL, который в свою очередь вызывает драйвер программа установки библиотек DLL и библиотеках translator установки они понадобятся. Установщик библиотеки DLL либо вызвать и непосредственно из панели управления или загружается и вызывается другой программой, известный как *программы администрирования*.
