# <a name="administration-program"></a>Программа администрирования
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включена в операционную систему Windows. ODBC следует только явным образом установить в более ранних версиях Windows.  
  
 Программа администрирования, а администратор ODBC, входит в состав Windows SDK/MDAC SDK. Этой программы и может свободно распространяться пользователями пакета SDK. Кроме того разработчики могут создавать собственные программы администрирования. Как правило разработчики пишут собственные программы администрирования, только в том случае, если они хотят сохранить полный контроль над конфигурации источника данных, или в том случае, если они настройки источников данных непосредственно из приложения, которое используется в качестве программы администрирования. Например электронных таблицах может позволить пользователям добавлять и затем использовать источники данных во время выполнения.  
  
 Программа администрирования сначала загружает библиотеки DLL установщика. Затем он вызывает функции в установщике DLL будет выполнять следующие задачи:  
  
-   **Добавить, изменить или удалить источники данных в интерактивном режиме.** Программа администрирования может вызывать **SQLManageDataSources**, **SQLCreateDataSource**, или **SQLConfigDataSource**.  
  
     **SQLManageDataSources** отображает диалоговое окно, с помощью которого пользователь может добавлять, изменять, или удалить источники данных и укажите параметры трассировки; эта функция вызывается, когда библиотека DLL установщика вызывается непосредственно из панели управления. **SQLCreateDataSource** отображает диалоговое окно, с помощью которого пользователь может только добавлять источники данных. **SQLConfigDataSource** передает этот вызов непосредственно к DLL-файлов установки драйвера.  
  
     Во всех случаях вызывает установщик DLL **ConfigDSN** в настройке драйвера библиотеки DLL для добавления, изменения или удаления источника данных. Библиотека DLL программы установки драйвера может запросить у пользователя дополнительную информацию.  
  
-   **Добавить, изменить или удалить источники данных без вмешательства пользователя.** Программа администрирования вызовет **SQLConfigDataSource** в библиотека DLL установщика и передает его окно null обработки, имя источника данных, чтобы добавить, изменить или удалить и список значений для раздела реестра. Вызовы библиотеки DLL установщика **ConfigDSN** в настройке драйвера библиотеки DLL для добавления, изменения или удаления источника данных.  
  
-   **Добавить, изменить или удалить источник данных по умолчанию.** Источник данных по умолчанию является таким же, как любой другой источник данных, за исключением того, что его имя: по умолчанию. Он будет добавлен, измененных или удаленных так же, как любой другой источник данных.
