# <a name="getapplicationintent-method-sqlserverdatasource"></a>Метод getApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение **applicationIntent** свойство соединения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает значение **applicationIntent** свойство соединения.  
  
## <a name="remarks"></a>Remarks  
 Дополнительные сведения о **applicationIntent** свойство подключения, см. в разделе [заданию свойств соединения](../../../connect/jdbc/setting-the-connection-properties.md).
