# <a name="concur_ss_scroll_locks-field-sqlserverresultset"></a>Поле CONCUR_SS_SCROLL_LOCKS (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Указывает тип оптимистичного параллелизма чтения-записи в [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] с блокировкой строки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public static final int CONCUR_SS_SCROLL_LOCKS  
```  
  
## <a name="field-value"></a>Значение поля  
 Значение **int** для 1009.
