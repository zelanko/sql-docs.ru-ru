# <a name="getclob-method-sqlservercallablestatement"></a>Метод getClob (SQLServerCallableStatement)

|[getClob (java.lang.String)](../../../connect/jdbc/reference/getclob-method-java-lang-string.md)|Получает значение указанного параметра JDBC BLOB в виде объекта Clob в языке программирования Java по заданному имени параметра.|
