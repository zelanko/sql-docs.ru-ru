# <a name="setbyte-method-sqlserverpreparedstatement"></a>setByte метод (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Устанавливает указанный параметр заданного **байтов** значение.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public final void setByte(int n,  
                          byte x)  
```  
  
#### <a name="parameters"></a>Параметры  
 *n*  
  
 **Int** указывает номер параметра.  
  
 *x*  
  
 Объект **байтов** значение.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод setByte указывается с помощью метода setByte в интерфейсе java.sql.PreparedStatement.
