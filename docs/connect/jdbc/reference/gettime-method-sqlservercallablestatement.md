# <a name="gettime-method-sqlservercallablestatement"></a>Метод getTime (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение указанного параметра в виде объекта java.sql.Time на языке программирования Java.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Описание|  
|----------|-----------------|  
|[Метод getTime (int)](../../../connect/jdbc/reference/gettime-method-int.md)|Получает значение указанного параметра в виде объекта java.sql.Time на языке программирования Java по заданному индексу параметра.|  
|[Метод getTime (int, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-int-java-util-calendar.md)|Возвращает значение заданного параметра в виде объекта java.sql.Time на языке программирования Java по индексу параметра, используя указанный объект Calendar.|  
|[Метод getTime (java.lang.String)](../../../connect/jdbc/reference/gettime-method-java-lang-string.md)|Получает значение указанного параметра в виде объекта java.sql.Time на языке программирования Java по заданному имени параметра.|  
|[Метод getTime (java.lang.String, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-java-lang-string-java-util-calendar.md)|Возвращает значение заданного параметра в виде объекта java.sql.Time на языке программирования Java по имени параметра, используя указанный объект Calendar.|
