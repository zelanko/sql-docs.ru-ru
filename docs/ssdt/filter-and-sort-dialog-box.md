# <a name="filter-and-sort-dialog-box"></a>Диалоговое окно фильтрации и сортировки
В обозревателе объектов SQL Server вы можете щелкнуть правой кнопкой мыши таблицу или представление и выбрать **Просмотреть данные**. См. дополнительные сведения в разделе [Как просмотреть и изменить данные в таблице](../ssdt/how-to-view-and-edit-data-in-a-table.md). Начиная с выпуска SQL Server Data Tools от марта 2014 г., в **редакторе данных** появилась кнопка сортировки и фильтрации на панели инструментов. Вы можете изменять данные, которые отображаются в таблице или представлении.  
  
В диалоговом окне **Фильтрация и сортировка** можно указать, какие данные будут отображаться в **редакторе данных**:  
  
-   количество отображаемых строк;  
  
-   столбцы для отображения;  
  
-   псевдонимы для имен столбцов;  
  
-   ограничения для отображения столбцов;  
  
-   порядок сортировки столбцов.  
  
Вы также можете щелкнуть правой кнопкой мыши строку в диалоговом окне **Фильтрация и сортировка** и выбрать операцию со строкой.
