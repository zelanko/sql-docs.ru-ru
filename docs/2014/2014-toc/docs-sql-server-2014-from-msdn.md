# <a name="documentation-for-sql-server-2014-is-moving-to-docs"></a>Документация на перенос документации по SQL Server 2014

В этой статье описывается предстоящим переходом статьями документации Microsoft SQL Server 2014. Эта статья посвящена в конце апреля 2018 г и перемещения произойдет перед июля 2018 г.

## <a name="links-to-sql-server-2014-documentation-on-msdn"></a>Ссылки на документацию SQL Server 2014 на MSDN

Ниже приведены ссылки для документации по SQL Server 2014, Microsoft хранились на его *MSDN* веб-сайта. До июля 2018 г., корпорация Майкрософт переместит статьях документации для ее нового *документация* веб-сайта:

- *SQL Server 2014*
    - [https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx](https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx)

- *Электронная документация по SQL Server 2014*
    - [https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx](https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx)

MSDN — это initialism для *Microsoft Developer Network*.


## <a name="redirectors"></a>Перенаправителями

Устаревшие адреса MSDN HTTP будут продолжать работать по крайней мере несколько лет после перемещения. Перенаправители будет перехватывать запросы MSDN и направляет их с их новый соответствующий адрес, на сайте docs.Microsoft.com. Тем не менее после перемещения перечисленные ниже, будут иметь значение true:

- Он будет лучше работать с нового адреса HTTP, которые перейти непосредственно на документы.
- Запустится программ-обходчиков Bing и Google delist в обход старые адреса MSDN.


## <a name="after-the-move"></a>После перемещения

После перехода на *документация*, статьи в документации SQL Server 2014 корневой адрес будет:

- [https://docs.microsoft.com/sql/?view=sql-server-2014](https://docs.microsoft.com/sql/?view=sql-server-2014)
