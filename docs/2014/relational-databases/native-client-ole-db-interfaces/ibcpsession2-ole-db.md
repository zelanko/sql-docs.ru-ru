# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  IBCPSession2, интерфейс является расширением IBCPSession, предоставляющий функцию-член, является альтернативой для вызова IBCPSession::BCPColFmt для каждого столбца.  IBCPSession2 наследует от интерфейса IBCPSession и добавлением одного нового метода: [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
