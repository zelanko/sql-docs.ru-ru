# <a name="setdisable-method-servernetworkprotocol-class"></a>Метод SetDisable (класс ServerNetworkProtocol)
  Отключает сетевой протокол сервера.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.SetDisable()  
  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 Объект [класс servernetworkprotocol Class] класс servernetworkprotocol-class.md), представляющий сетевой протокол, используемый экземпляром [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Значение uint32, равное 0, если служба изменена успешно, и 1, если запрос не поддерживается. Любое другое значение указывает на ошибку.
