# <a name="protocolname-property-servernetworkprotocol-class"></a>Свойство ProtocolName (класс ServerNetworkProtocol)
  Возвращает имя сетевого протокола сервера.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 A [класса ServerNetworkProtocol](servernetworkprotocol-class.md) , который представляет сетевой протокол, используемый экземпляром [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строка, определяющая имя сетевого протокола сервера.
