# <a name="protocolname-property-servernetworkprotocolipaddress-class"></a>Свойство ProtocolName (класс ServerNetworkProtocolIPAddress)
  Возвращает имя сетевого протокола сервера, ссылающееся на IP-адрес.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 A [класса ServerNetworkProtocolIPAdress](servernetworkprotocolipaddress-class.md) , который представляет IP-адрес сетевого протокола для экземпляра [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, определяющее имя сетевого протокола сервера.
