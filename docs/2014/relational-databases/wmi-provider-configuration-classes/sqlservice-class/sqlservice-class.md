# <a name="sqlservice-class"></a>Класс SqlService
  Класс [SqlService](sqlservice-class.md) представляет экземпляр службы [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
 Класс [SqlService](sqlservice-class.md) используется для программного управления службой [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] . Работая с объектом класса [SqlService](sqlservice-class.md) , можно выполнять следующие задачи:  
  
-   Подключение к службе.  
  
-   изменять учетную запись и пароль, которые используются для соединения со службой;  
  
-   запускать, останавливать, приостанавливать и возобновлять выполнение службы;  
  
-   проверять свойства службы.
