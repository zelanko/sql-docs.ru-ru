# <a name="instancename-property-wmi-msreportserver_configurationsetting"></a>Свойство InstanceName (WMI MSReportServer_ConfigurationSetting)
  Указывает имя экземпляра сервера отчетов на заданном компьютере. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## <a name="property-value"></a>Значение свойства  
 Объект `String`, представляющий имя экземпляра сервера отчетов.
