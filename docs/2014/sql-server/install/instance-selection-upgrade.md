# <a name="instance-selection-upgrade"></a>Выбор экземпляра (обновление)
  Используйте страницу «Выбор экземпляра (обновление)» для указания обновляемого экземпляра [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="options"></a>Параметры  
 Все компоненты экземпляра должны быть обновлены одновременно. Добавить или удалить компоненты в процессе обновления нельзя.
