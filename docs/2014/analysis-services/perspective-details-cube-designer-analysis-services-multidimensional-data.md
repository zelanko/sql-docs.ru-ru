# <a name="perspective-details-perspectives-tab-cube-designer-analysis-services---multidimensional-data"></a>Подробности перспективы (вкладка «Перспективы», конструктор кубов) (службы Analysis Services — многомерные данные)
  С помощью панели **Подробности перспективы** на вкладке **Перспективы** конструктора кубов можно управлять метаданными, доступными пользователям, запрашивающим выбранную перспективу.  
  
## <a name="options"></a>Варианты  
 **Объекты куба**  
 Отображается иерархический список объектов и свойств, содержащихся в кубе, который может быть включен в перспективу.  
  
 **Тип объекта**  
 Отображается тип или имя свойства объекта из куба, который может быть включен в перспективу.  
  
 **Имя перспективы**  
 Отображаются значения свойств и данные о том, включен ли объект куба в перспективу, представленную столбцом **Имя перспективы** . Для каждой перспективы в кубе содержится один столбец **Имя перспективы** .  
  
 Введите имя перспективы в ячейку для свойства **Имя** куба для выбранной перспективы.  
  
 Выберите меру по умолчанию перспективы в ячейке для свойства **DefaultMeasure** куба для выбранной перспективы.  
  
 Выберите объект, чтобы включить его в перспективу.  
  
## <a name="context-menu"></a>Контекстное меню  
 Следующие пункты доступны в контекстном меню, которое появляется, если щелкнуть правой кнопкой мыши любую ячейку перспективы на панели **Подробности перспективы** .  
  
|Параметр|Описание|  
|------------|-----------------|  
|**Создать перспективу**|Выберите этот пункт, чтобы создать новую перспективу в выбранном кубе.|  
|**Удалить перспективу**|Нажмите, чтобы отобразить диалоговое окно **Удаление объектов** и удалить выбранную перспективу|
