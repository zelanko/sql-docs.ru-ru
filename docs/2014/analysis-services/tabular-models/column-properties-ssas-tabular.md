# <a name="column-properties-ssas-tabular"></a>Свойства столбца (табличные службы SSAS)
  В этом разделе описаны свойства столбца табличной модели.  
  
 Разделы данной темы:  
  
-   [Свойства столбца](#bkmk_properties)  
  
-   [Настройка параметров свойств столбца](#bkmk_config_prop)  
  
##  <a name="bkmk_properties"></a> Свойства столбца  
 **Базовая**  
  
|Свойство|Параметр по умолчанию|Описание|  
|--------------|---------------------|-----------------|  
|**Имя столбца**||Имя столбца в виде, сохраненном в модели, и в виде, отображаемом в списке полей клиента отчетов.|  
|**Формат данных**|Автоматически определяется во время импорта.|Указывает формат отображения для данных в этом столбце. После установки формата данных можно установить свойства, характерные для каждого формата отдельно. Например, при выборе формата **Валюта** можно установить количество видимых разрядов после запятой, выбрать разделитель на тысячные доли, а также выбрать символ валюты. Это свойство имеет следующие параметры.<br /><br /> **Общие сведения**<br /><br /> **Decimal Number**<br /><br /> **Whole Number**<br /><br /> **Измерение валют**<br /><br /> **Процент**<br /><br /> **Научный**<br /><br /> Если значения столбца содержат изображения, см. раздел **Представительное изображение**.|  
|**Тип данных**|Автоматически определяется во время импорта.|Указывает тип данных для всех значений в столбце.|  
|**Описание**||Текстовое описание для столбца.<br /><br /> В некоторых клиентских средствах создания отчетов, когда пользователь наводит указатель на этот столбец в списке полей, описание отображается в виде подсказки.|  
|**Скрыта**|False|Указывает, является ли столбец скрытым при выводе отчета о списке полей клиента.<br /><br /> Установите это свойство в значение **True** , чтобы столбец не отображался. Это можно сделать для столбцов, содержащих идентификаторы или ключи, которые обычно не нужны конечному пользователю.<br /><br /> При скрытии столбцов от клиента отчетов указанное поле не подавляется для данных модели. Поле будет по-прежнему видно при создании запроса для модели. Скрытый столбец может как и раньше использоваться для группирования или сортировки.<br /><br /> Свойство **Скрытый** не связано ни с одним из способов обеспечения безопасности. Для защиты данных используйте фильтры строк в ролях. Дополнительные сведения см. в разделе [Roles &#40;SSAS Tabular&#41;](roles-ssas-tabular.md).|  
|**Сортировать по столбцам**||Определяет другой столбец для сортировки значений в этом столбце. Между двумя столбцами должна существовать связь.<br /><br /> Это значение должно быть именем существующего столбца. Нельзя указывать формулу или меру.|  
  
 **Свойства отчетов**  
  
 Дополнительные сведения о настройке свойств поведения таблицы [!INCLUDE[ssCrescent](../../includes/sscrescent-md.md)] см. в разделе [Настройка свойств работы таблицы для отчетов Power View (табличные службы SSAS)](power-view-configure-table-behavior-properties-for-reports.md).  
  
|Свойство|Параметр по умолчанию|Описание|  
|--------------|---------------------|-----------------|  
|Изображение по умолчанию|False|Указывает, в каком столбце содержится изображение с данными строки (например, идентификатор фотографии в записи о сотруднике).|  
|Метка по умолчанию|False|Указывает, в каком столбце содержится отображаемое имя, представляющее данные строки (например, имя сотрудника в записи о сотруднике).|  
|Категория URL-адреса или данных изображения (пакет обновления 1 (SP1)|False|Указывает значение в данном столбце как гиперссылку на изображение на сервере. Например, http://localhost/images/image1.jpg.|  
|Сохранять уникальные строки|False|Указывает, в каких столбцах содержатся значения, которые должны обрабатываться как уникальные, даже если они являются дубликатами (например, имя и фамилия сотрудника, если несколько сотрудников имеют одинаковые имена).|  
|Идентификатор строки|False|Указывает столбец, содержащий только уникальные значения, что дает возможность использовать этот столбец в качестве внутреннего ключа группирования.|  
|Суммировать по|По умолчанию|Указывает, что клиентские средства составления отчетов применяют агрегатную функцию SUM для вычисления столбцов, когда этот столбец добавляется в список полей. Чтобы изменить вычисление по умолчанию, выберите его из раскрывающегося списка. Это свойство применяется только к столбцам, тип которых поддерживает статистическую обработку.|  
|Позиция таблицы со сведениями|Поле по умолчанию не задано|Указывает, что этот столбец или меру можно добавить в набор полей из одной таблицы, чтобы улучшить представление таблицы в клиенте для работы с отчетами.|  
  
###  <a name="bkmk_config_prop"></a> Настройка параметров свойств столбца  
  
1.  В конструкторе моделей в таблице выберите столбец.  
  
2.  В окне **Свойства** щелкните свойство и введите значение или нажмите стрелку вниз для выбора параметра.
