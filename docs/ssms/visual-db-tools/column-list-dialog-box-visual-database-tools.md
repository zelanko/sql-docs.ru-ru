# <a name="column-list-dialog-box-visual-database-tools"></a>Диалоговое окно «Список столбцов» (визуальные инструменты для баз данных)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
При написании запросов полезно иметь перед собой список столбцов, доступных для выбранных табличных объектов.  
  
Чтобы открыть это диалоговое окно, в меню **Конструктор запросов** на панели "Диаграмма" щелкните правой кнопкой мыши заголовок табличного объекта и выберите команду **Свойства**. В окне **Свойства** щелкните поле **Список столбцов** . Затем нажмите кнопку с многоточием **(...)** справа от этого поля.  
  
В данном диалоговом окне отображается список столбцов в выбранном табличном объекте и такие основные свойства, как типы данных и возможность присвоения значений NULL.  
  
## <a name="options"></a>Параметры  
**Название**  
Имена столбцов в выбранном табличном объекте.  
  
**Тип данных**  
Типы данных каждого столбца в выбранном табличном объекте.  
  
**Свойства типа данных**  
Свойства типов данных каждого столбца в выбранном табличном объекте. Возможные значения: NULLABLE, IDENTITY, ROWGUID и пустое значение.
