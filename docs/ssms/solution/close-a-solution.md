# <a name="close-a-solution"></a>Закрытие решения
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Решение можно закрыть и удалить его из обозревателя решений. После закрытия решения использовать его в обозревателе решений будет невозможно.  
  
> [!NOTE]  
> Закрытие обозревателя решений не закрывает решение автоматически. Оно лишь скрывает окно обозревателя решений.  
  
### <a name="to-close-a-solution"></a>Закрытие решения  
  
1.  В меню **Файл** выберите пункт **Закрыть решение**.
