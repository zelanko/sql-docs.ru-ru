# <a name="add-or-delete-a-group-in-a-data-region-report-builder-and-ssrs"></a>Добавление или удаление группы в области данных (построитель отчетов и службы SSRS)
В отчетах [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] с разбиением на страницы группа добавляется к области данных для организации данных в зависимости от указанного значения или набора выражений, для отображения и вычислений. Группа имеет имя и выражение, указывающее, какие данные из набора принадлежат к этой группе. Дополнительные сведения о группах см. в разделе [Основные сведения о группах (построитель отчетов и службы SSRS)](../../reporting-services/report-design/understanding-groups-report-builder-and-ssrs.md).  
  
 В области данных табликса щелкните таблицу, матрицу или список, чтобы вывести панель группирования. Перетащите поля из набора данных в панель «Группы строк» или «Группы столбцов» для создания родительских или дочерних групп. Щелкните правой кнопкой мыши существующую группу, чтобы добавить смежную с ней группу. По определению группа подробностей является самой внутренней группой и может быть добавлена только как дочерняя. Щелкните правой кнопкой мыши существующую группу, чтобы удалить ее. Строки и столбцы для вывода значений группы добавляются автоматически. Дополнительные сведения см. в разделе [Таблицы, матрицы и списки (построитель отчетов и службы SSRS)](../../reporting-services/report-design/tables-matrices-and-lists-report-builder-and-ssrs.md).  
  
 Щелкните в области данных диаграммы, чтобы отобразить области перетаскивания. Создайте группы, перетащив поля набора данных на области перетаскивания категории и рядов. Чтобы добавить вложенные группы, в область перетаскивания добавляют несколько полей.  
  
 Группы не создаются в датчике по умолчанию. По умолчанию датчик только вычисляет в указанном поле статистический показатель всех значений и выводит эту величину на датчике. Дополнительные сведения см. в разделе [Датчики (построитель отчетов и службы SSRS)](../../reporting-services/report-design/gauges-report-builder-and-ssrs.md).  
  
> [!NOTE]  
>  [!INCLUDE[ssRBRDDup](../../includes/ssrbrddup-md.md)]  
  
## <a name="to-add-a-parent-or-child-row-or-column-group-to-a-tablix-data-region"></a>Добавление дочерней или родительской группы строк или столбцов в область данных табликса  
  
1.  Перетащите поле из области **Данные отчета** на область **Группы рядов** или область **Группы столбцов** .  
  
    > [!NOTE]  
    >  Если панель группирования не видна, на вкладке "Вид" выберите пункт **Группирование**.  
  
2.  Перетащите поле и отпустите его над или под иерархией групп, используя панель консультанта, чтобы разместить эту группу как родительскую или дочернюю по отношению к существующей группе.  
  
     Группа добавляется с именем по умолчанию, выражением группы и выражением сортировки на основании имени поля.  
  
## <a name="to-add-an-adjacent-row-or-column-group-to-a-tablix-data-region"></a>Добавление смежной группы строк или столбцов в область данных табликса  
  
1.  На панели группирования щелкните правой кнопкой мыши группу, одноранговую добавляемой группе. Щелкните **Добавить группу**, а затем **Прилегающая до** или **Прилегающая после** , чтобы указать, куда именно добавить группу. Откроется диалоговое окно **Группа табликсов** .  
  
2.  В поле **Имя**введите имя группы.  
  
3.  Чтобы создать выражение, в поле **Выражение группирования**нажмите кнопку (**fx**) или введите новое выражение.  
  
4.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
     На панель группирования будет добавлена новая группа, а в область данных табликса в области конструктора будут добавлены строка или столбец для вывода групповых значений.  
  
## <a name="to-add-a-details-group-to-a-tablix-data-region"></a>Добавление группы сведений к области данных табликса  
  
1.  На панели "Группирование" щелкните правой кнопкой мыши самую внутреннюю дочернюю группу, но не группу **Сведения** . Нажмите **Добавить группу**и выберите **Дочерняя группа**. Откроется диалоговое окно **Группа табликсов** .  
  
2.  Оставьте поле **Выражение группы**пустым. Группа подробностей не имеет выражения.  
  
3.  Выберите **Показать подробные данные**.  
  
4.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
     Новая группа подробностей добавляется в панель «Группирование» в виде дочерней группы, и маркер строки для группы, выбранной в шаге 1, отображает значок группы подробностей. Дополнительные сведения о маркерах см. в разделе [Ячейки, строки и столбцы области данных табликса &#40;построитель отчетов&#41; и службы SSRS](../../reporting-services/report-design/tablix-data-region-cells-rows-and-columns-report-builder-and-ssrs.md).  
  
## <a name="to-edit-a-row-or-column-group-in-a-tablix-data-region"></a>Изменение группы строк или столбцов в области данных табликса  
  
1.  В области конструктора щелкните в любом месте области данных табликса для ее выделения. На панели группирования будут отображены группы столбцов и строк.  
  
2.  Щелкните правой кнопкой мыши группу, а затем щелкните **Свойства группы**.  
  
3.  В поле **Имя**введите имя группы.  
  
4.  Чтобы создать выражение группирования, в поле **Выражения группирования**введите или выберите простое выражение или нажмите кнопку (**fx**).  
  
5.  Для создания дополнительных выражений щелкните **Добавить** . Все заданные выражения будут соединены с помощью логического оператора ИЛИ, чтобы образовать выражение группирования данной группы.  
  
6.  Перейдите на вкладку **Разрывы страниц** , чтобы задать параметры разрывов страниц (необязательно).  
  
7.  Перейдите на вкладку **Сортировка** , чтобы выбрать или ввести выражение для сортировки данных внутри группы (необязательно).  
  
8.  Перейдите на вкладку **Видимость** , чтобы выбрать параметры видимости для элемента (необязательно).  
  
9. Перейдите на вкладку **Фильтры** , чтобы задать фильтры для группы (необязательно).  
  
10. Перейдите на вкладку **Переменные** , чтобы задать переменные, областью действия которых будет данная группа и все ее дочерние группы (необязательно).  
  
11. [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
## <a name="to-delete-a-group-from-a-tablix-data-region"></a>Удаление группы из области данных табликса  
  
1.  На панели "Группирование" щелкните правой кнопкой мыши группу и выберите команду **Удалить группу**.  
  
2.  В диалоговом окне **Удаление группы** выберите один из следующих параметров:  
  
    -   **Удалить группу, а также связанные строки и столбцы** Этот параметр удаляет определение группы и все связанные с ней строки, в которых выводятся данные группы. В группе подробностей, если одна и та же строка принадлежит одновременно подробностям и данным группы, удаляются только строки данных подробностей.  
  
    -   **Удалить только группу** . Если выбрать этот параметр, область данных табликса сохранит свою структуру. Удалено будет только определение группы.  
  
3.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
## <a name="to-delete-a-details-group-from-a-tablix-data-region"></a>Удаление группы подробностей из области данных табликса  
  
1.  На панели "Группирование" щелкните правой кнопкой мыши группу подробностей и выберите команду **Удалить группу**.  
  
2.  В диалоговом окне **Удаление группы** выберите один из следующих параметров:  
  
    -   **Удалить группу, а также связанные строки и столбцы** Этот параметр удаляет определение группы и все связанные с ней строки, в которых выводятся данные группы. В группе подробностей, если одна и та же строка принадлежит одновременно подробностям и данным группы, удаляются только строки данных подробностей.  
  
    -   **Удалить только группу** . Если выбрать этот параметр, область данных табликса сохранит свою структуру. Удалено будет только определение группы.  
  
3.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
     Группа подробностей будет удалена.  
  
    > [!NOTE]  
    >  Убедитесь, что после удаления строки подробностей выражение в каждой ячейке содержит статистическое выражение (там, где это уместно). При необходимости измените выражение, чтобы задать агрегатные функции нужным образом.
