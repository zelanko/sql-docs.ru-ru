# <a name="sortorderenum"></a>SortOrderEnum
Указывает последовательность сортировки для индексированного столбца.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|По умолчанию. Последовательности сортировки для столбца по возрастанию.|  
|**adSortDescending**|2|Последовательности сортировки для столбца — по убыванию.|
