# <a name="errorvalueenum"></a>ErrorValueEnum
Указывает тип ошибки времени выполнения ADO.  
  
 Ниже перечислены три формы номера ошибки.  
  
-   Положительное десятичное число — младшие два байта полного числа в десятичном формате. Это число отображается в диалоговом окне Visual Basic сообщение об ошибке по умолчанию. Например, ошибка времени выполнения "3707".  
  
-   Отрицательное десятичное число — десятичное преобразование полного номера ошибки.  
  
-   Шестнадцатеричное — шестнадцатеричное представление полного номера ошибки. Код устройства Windows находится в четвертой цифре. Код устройства для номеров ошибок ADO — *это*. Например: 0x800***A***0E7B.  
  
> [!NOTE]
>  OLE DB ошибки могут передаваться в приложение ADO. Как правило, их можно идентифицировать с помощью кода оборудования Windows *4*. Например, 0x800***4***.
