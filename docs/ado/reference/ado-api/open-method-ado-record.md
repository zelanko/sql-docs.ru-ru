# <a name="open-method-ado-record"></a>Метод Open (объект Record ADO)
Открывает существующий объект [Record](./record-object-ado.md) или создает новый элемент, представленный **записью**, например файл или каталог.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Open Source, ActiveConnection, Mode, CreateOptions, Options, UserName, Password  
```  
  
#### <a name="parameters"></a>Параметры  
 *Source*  
 Необязательный элемент. **Вариант** , который может представлять URL-адрес сущности, представляемый этим объектом **записи** , **команду**, Открытый [набор записей](./recordset-object-ado.md) или другой объект **записи** , строку, содержащую инструкцию SQL SELECT или имя таблицы.  
  
 *ActiveConnection*  
 Необязательный элемент. **Вариант** , представляющий строку подключения или открытый объект [соединения](./connection-object-ado.md) .  
  
 *Режим*  
 Необязательный элемент. Значение [коннектмодинум](./connectmodeenum.md) , указывающее режим доступа для результирующего объекта **записи** . Значение по умолчанию — **адмодеункновн**.  
  
 *креатеоптионс*  
 Необязательный элемент. Значение [рекордкреатеоптионсенум](./recordcreateoptionsenum.md) , указывающее, следует ли открывать существующий файл или каталог или создавать новый файл или каталог. Значение по умолчанию — **адфаилифнотексистс**. Если задано значение по умолчанию, режим доступа получается из свойства [mode](./mode-property-ado.md) . Этот параметр игнорируется, если параметр *Source* не содержит URL-адрес.  
  
 *Параметры*  
 Необязательный элемент. Значение [рекордопеноптионсенум](./recordopenoptionsenum.md) , указывающее параметры для открытия **записи**. Значение по умолчанию — **адопенрекордунспеЦифиед**. Эти значения можно комбинировать.  
  
 *UserName*  
 Необязательный элемент. **Строковое** значение, содержащее идентификатор пользователя, который, при необходимости, авторизует доступ к *источнику*.  
  
 *Пароль*  
 Необязательный элемент. **Строковое** значение, содержащее пароль, который при необходимости проверяет *имя пользователя*.  
  
## <a name="remarks"></a>Remarks  
 *Источником* может быть:  
  
-   URL-адрес. Если для URL-адреса используется протокол HTTP, поставщик услуг Интернета будет вызываться по умолчанию. Если URL-адрес указывает на узел, содержащий исполняемый сценарий (например,. ASP-страница), по умолчанию открывается **запись** , содержащая источник, а не выполненное содержимое. Чтобы изменить это поведение, используйте аргумент *Options* .  
  
-   Объект **Record** . Объект **записи** , Открытый из другой **записи** , будет клонировать исходный объект **записи** .  
  
-   Объект **команды** . Открытый объект **записи** представляет одну строку, возвращенную при помощи **команды**. Если результаты содержат более одной строки, содержимое первой строки помещается в запись, а в коллекцию **Errors** может быть добавлена ошибка.  
  
-   Инструкция SQL SELECT. Открытый объект **записи** представляет одну строку, возвращенную путем выполнения содержимого строки. Если результаты содержат более одной строки, содержимое первой строки помещается в запись, а в коллекцию **Errors** может быть добавлена ошибка.  
  
-   Имя таблицы.  
  
 Если объект **Record** представляет сущность, доступ к которой невозможен с помощью URL-адреса (например, строки **набора записей** , производного от базы данных), то значения свойства [парентурл](./parenturl-property-ado.md) и поля, к которым обращается константа **адрекордурл** , имеют значение null.  
  
> [!NOTE]
>  URL-адреса, использующие схему HTTP, автоматически вызывают [поставщик OLE DB Майкрософт для публикации в Интернете](../../guide/appendixes/microsoft-ole-db-provider-for-internet-publishing.md). Дополнительные сведения см. в разделе [абсолютные и относительные URL-адреса](../../guide/data/absolute-and-relative-urls.md).
