# <a name="fieldattributeenum"></a>FieldAttributeEnum
Задает один или несколько атрибутов объекта [field](../../../ado/reference/ado-api/field-object.md) .  
  
|Константа|Применение|Описание|  
|--------------|-----------|-----------------|  
|**adFldCacheDeferred**|0x1000|Указывает, что поставщик кэширует значения полей и что последующие операции чтения выполняются из кэша.|  
|**адфлдфиксед**|0x10|Указывает, что поле содержит данные фиксированной длины.|  
|**adFldIsChapter**|0x2000|Указывает, что поле содержит значение главы, которое указывает конкретный дочерний набор записей, связанный с этим родительским полем. Обычно поля глав используются с формированием данных или фильтрами.|  
|**адфлдисколлектион**|0x40000|Указывает, что в поле указано, что ресурс, представленный записью, представляет собой коллекцию других ресурсов, например папку, а не простой ресурс, например текстовый файл.|  
|**adFldKeyColumn**|0x8000|Указывает, что поле указывает весь первичный ключ столбца или его часть.|  
|**adFldIsDefaultStream**|0x20000|Указывает, что поле содержит поток по умолчанию для ресурса, представленного записью. Например, поток по умолчанию может быть содержимым HTML корневой папки на веб-сайте, которое автоматически обрабатывается при указании корневого URL-адреса.|  
|**adFldIsNullable**|0x20|Указывает, что поле принимает значения NULL.|  
|**adFldIsRowURL**|0x10000|Указывает, что поле содержит URL-адрес, который именует ресурс из хранилища данных, представленного записью.|  
|**адфлдлонг**|0x80|Указывает, что поле является длинным бинарным полем. Также указывает, что вы можете использовать методы [AppendChunk](../../../ado/reference/ado-api/appendchunk-method-ado.md) и " [блока](../../../ado/reference/ado-api/getchunk-method-ado.md) ".|  
|**adFldMayBeNull**|0x40|Указывает, что можно считать значения NULL из поля.|  
|**adFldMayDefer**|0x2|Указывает, что поле откладывается, то есть значения полей не извлекаются из источника данных с целой записью, но только при явном доступе к ним.|  
|**адфлднегативескале**|0x4000|Указывает, что поле представляет числовое значение из столбца, поддерживающего отрицательные значения шкалы. Масштаб задается свойством [NumericScale](../../../ado/reference/ado-api/numericscale-property-ado.md) .|  
|**adFldRowID**|0x100|Указывает, что поле содержит постоянный идентификатор строки, который не может быть записан в и не имеет осмысленного значения, кроме идентификации строки (например, номера записи, уникального идентификатора и т. д.).|  
|**adFldRowVersion**|0x200|Указывает, что поле содержит некоторую метку времени или даты, используемую для наблюдения за обновлениями.|  
|**адфлдункновнупдатабле**|0x8|Указывает, что поставщик не может определить, можно ли выполнить запись в поле.|  
|**адфлдунспеЦифиед**|-1 0xFFFFFFFF|Указывает, что поставщик не задает атрибуты поля.|  
|**адфлдупдатабле**|0x4|Указывает, что можно выполнить запись в поле.|
