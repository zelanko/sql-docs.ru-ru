# <a name="records-and-provider-supplied-fields"></a>Записи и предоставляемые поставщиком поля
При [записи](../../../ado/reference/ado-api/record-object-ado.md) объект открыт, ее источник может быть текущей строки открытого [записей](../../../ado/reference/ado-api/recordset-object-ado.md), абсолютный URL-адрес или относительный URL-адрес в сочетании с открытым подключением [подключения](../../../ado/reference/ado-api/connection-object-ado.md) объекта .  
  
 Если **записи** открывается из **записей**, **записи** объект [поля](../../../ado/reference/ado-api/fields-collection-ado.md) коллекция будет содержать все поля из  **Набор записей**, плюс все поля, добавленные базового поставщика.  
  
 Поставщик может вставить дополнительные поля, которые служат дополнительные характеристики **записи**. Таким образом **записи** не может иметь уникальные поля **записей** как целое или любой **записи** производным от другой строке **записей**.  
  
 Например, все строки **записей** производные в сообщении электронной почты, источник данных может содержать столбцы, например, в, чтобы и субъектом. Объект **записи** производный от класса, **записей** будет иметь те же поля. Тем не менее **записи** также могут быть другие поля, уникальные для определенного сообщения, представленный, **записи**, таких как вложения и Cc (копия).  
  
 Несмотря на то что **записи** объекта и текущей строке **записей** имеют те же поля, они отличаются поскольку **записи** и **записей**объекты имеют различные методы и свойства.  
  
 Поле общего удерживаемые **записи** и **записей** можно изменить на любой из этих объектов. Тем не менее, поле не может быть удалено на **записи** объекта, несмотря на то, что базовый поставщик может поддерживать, установив в поле значение null.  
  
 После **записи** будет открыт, можно программно добавить поля. Вы также можете удалить поля, которые были добавлены, но не может удалить поля из исходного **записей**.  
  
 Можно также открыть **записи** объект непосредственно из URL-адрес. В этом случае поля добавлены **записи** зависят от базового поставщика. В настоящее время большинство поставщиков Добавление набора полей, которые описывают сущности, представленной **записи**. Если сущность состоит из потока байтов, например файл простой [Stream](../../../ado/reference/ado-api/stream-object-ado.md) объект обычно может быть открыто из **записи**.  
  
## <a name="special-fields-for-document-source-providers"></a>Поставщики версиями специальные поля для документа  
 Особый класс поставщиков, вызывается *документа источника поставщики*, управление папками и документы. Когда **записи** представляет объект документа или в **записей** представляет папку документов, поставщик источника документа заполняет объекты с помощью уникального набора полей, которые описывают характеристики документа вместо реального самого документа. Как правило, одно поле содержит ссылку на **Stream** , представляющий документ.  
  
 Эти поля образуют ресурс **записи** или **записей** и перечислены для отдельных поставщиков, поддерживающих их в [приложении a. Поставщики](../../../ado/guide/appendixes/appendix-a-providers.md).  
  
 Индекс две константы **поля** коллекцию ресурс **записи** или **записей** для получения пары часто используемых полей. **Поле** объект [значение](../../../ado/reference/ado-api/value-property-ado.md) свойство возвращает нужное содержимое.  
  
-   Поле, для доступа к **adDefaultStream** константа содержит поток по умолчанию, связанные с **записи** или **записей** объекта. Поставщик назначает поток по умолчанию для объекта.  
  
-   Поле, для доступа к **adRecordURL** константа содержит абсолютный URL-адрес для идентификации документа.  
  
 Не поддерживает поставщик источника документа [свойства](../../../ado/reference/ado-api/properties-collection-ado.md) коллекцию **записи** и **поле** объектов. Содержание **свойства** коллекция имеет значение null, для таких объектов.  
  
 Поставщик источника документа может добавить свойства поставщика, например **тип источника данных** для определения, является ли поставщик исходного документа. Дополнительные сведения о том, как определение типа поставщика, см. в документации поставщика.  
  
## <a name="resource-recordset-columns"></a>Столбцы наборов записей ресурсов  
 Объект *записей ресурсов* состоит из следующих столбцов.  
  
|Имя столбца|Тип|Описание|  
|-----------------|----------|-----------------|  
|RESOURCE_PARSENAME|adVarWChar|Только для чтения. Указывает URL-адрес ресурса.|  
|RESOURCE_PARENTNAME|adVarWChar|Только для чтения. Указывает абсолютный URL-адрес родительской записи.|  
|RESOURCE_ABSOLUTEPARSENAME|adVarWChar|Только для чтения. Указывает абсолютный URL-адрес ресурса, который является объединение PARENTNAME и PARSENAME.|  
|RESOURCE_ISHIDDEN|adBoolean|Значение true, если ресурс является скрытым. Строки не возвращаются, если команде, которая создает набор строк явным образом выбирает строки, где RESOURCE_ISHIDDEN имеет значение True.|  
|RESOURCE_ISREADONLY|adBoolean|Значение true, если ресурс доступен только для чтения. Пытается открыть этот ресурс с DBBINDFLAG_WRITE и будет завершаться DB_E_READONLY. Это свойство можно изменить даже в том случае, если ресурс был открыт только для чтения.|  
|RESOURCE_CONTENTTYPE|adVarWChar|Указывает вероятный вариант использования документа — например, юрист краткое. Это может соответствовать шаблон Office, который использовался для создания документа.|  
|RESOURCE_CONTENTCLASS|adVarWChar|Указывает MIME-тип документа, например, указывающее формат "`text/html`«.|  
|RESOURCE_CONTENTLANGUAGE|adVarWChar|Указывает язык, в которой хранится содержимое.|  
|RESOURCE_CREATIONTIME|adFileTime|Только для чтения. Указывает на структуру FILETIME, содержит время создания ресурса. Время указывается в формате времени (UTC).|  
|RESOURCE_LASTACCESSTIME|adFileTime|Только для чтения. Указывает на структуру FILETIME, содержащий время последнего доступа к ресурсу. Время представлено в формате UTC. Члены FILETIME равны нулю, если поставщик не поддерживает этот элемент времени.|  
|RESOURCE_LASTWRITETIME|adFileTime|Только для чтения. Указывает на структуру FILETIME, которая содержит время последней операции записи ресурса. Время представлено в формате UTC. Члены FILETIME равны нулю, если поставщик не поддерживает этот элемент времени.|  
|RESOURCE_STREAMSIZE|asUnsignedBigInt|Только для чтения. Указывает размер потока ресурса по умолчанию, в байтах.|  
|RESOURCE_ISCOLLECTION|adBoolean|Только для чтения. Значение true, если ресурс является коллекцией, например в каталоге. Значение false, если ресурс является простой файл.|  
|RESOURCE_ISSTRUCTUREDDOCUMENT|adBoolean|Значение true, если ресурс является структурированного документа. Значение false, если ресурс не структурированного документа. Это может быть коллекция или простой файл.|  
|DEFAULT_DOCUMENT|adVarWChar|Только для чтения. Указывает, что этот ресурс содержит URL-адрес простой документ по умолчанию папки или структурированного документа. Используется при запросе потока по умолчанию из ресурса. Это свойство будет пустым для простой файл.|  
|CHAPTERED_CHILDREN|adChapter|Только для чтения. Необязательный. Указывает главы из набора строк, содержащий дочерний ресурс. ( *Поставщик OLE DB для публикаций в Интернете* не использовали этот столбец.)|  
|RESOURCE_DISPLAYNAME|adVarWChar|Только для чтения. Указывает отображаемое имя ресурса.|  
|RESOURCE_ISROOT|adBoolean|Только для чтения. Значение true, если ресурс является корнем коллекции или структурированного документа.|
