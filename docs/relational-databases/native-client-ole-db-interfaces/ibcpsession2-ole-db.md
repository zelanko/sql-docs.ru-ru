# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Интерфейс IBCPSession2 является расширением интерфейса IBCPSession, который предоставляет функцию-элемент, что является альтернативой вызову интерфейса IBCPSession::BCPColFmt для каждого столбца.  IBCPSession2 наследует от IBCPSession и добавляет один новый метод: [IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
