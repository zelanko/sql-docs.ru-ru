# <a name="mssqlserver2593"></a>MSSQLSERVER_2593
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>Сведения  
  
|||  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|2593|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|DBCC_OBJECT_ROW_PAGE_SUMMARY|  
|Текст сообщения|Имеется ROWCOUNT, строк на PAGECOUNT страницах для объекта «OBJECT».|  
  
## <a name="explanation"></a>Объяснение  
Это сообщение является частью информационного вывода, который возвращается всеми проверками DBCC, за исключением инструкции DBCC CHECKALLOC, и указывает число строк и страниц для указанного объекта.  
  
## <a name="user-action"></a>Действие пользователя  
None
