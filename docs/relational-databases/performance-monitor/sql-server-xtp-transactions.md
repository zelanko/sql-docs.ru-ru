# <a name="sql-server-xtp-transactions"></a>XTP-транзакции SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  Объект производительности транзакций XTP SQL Server содержит счетчики, относящиеся к транзакциям выполняющейся в памяти OLTP в [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 В этой таблице перечислены счетчики **транзакций XTP SQL Server** .  
  
|Счетчик|Description|  
|-------------|-----------------|  
|**Каскадных прерываний/с**|Количество транзакций, для которых был сделан откат из-за отката зависимостей фиксации (в среднем), в секунду.|  
|**Зависимостей фиксации принято/с**|Общее число зависимостей фиксации, принятых транзакциями (в среднем) в секунду.|  
|**Транзакций в режиме только для чтения подготовлено/с**|Количество транзакций только для чтения, подготовленных для обработки фиксации, в секунду.|  
|**Обновлений точек сохранения/с**|Количество случаев, когда точка сохранения была «обновлена» (в среднем), в секунду. Обновление точки сохранения называется сбросом существующей точки сохранения к текущей точке за время существования транзакции.|  
|**Откатов точек сохранения/с**|Количество раз, когда произошел откат транзакции к точке сохранения (в среднем), в секунду.|  
|**Создано точек сохранения за секунду**|Число созданных точек сохранения (в среднем) в секунду.|  
|**Ошибок проверки транзакций/с**|Общее число транзакций, которые не удалось проверить (в среднем), в секунду.|  
|**Транзакций, прерванных пользователем/с**|Общее число транзакций, которые были прерваны пользователем (в среднем), в секунду.|  
|**Прервано транзакций/с**|Число прерванных (как пользователем, так и системой, в среднем) транзакций в секунду.|  
|**Создано транзакций/с**|Количество транзакций, созданных в системе (в среднем), в секунду<br /><br /> Транзакции XTP подсчитываются не так, как дисковые транзакции (как отражено в базах данных: транзакций/с). Например, параметр «Созданные транзакции/с» учитывает транзакции чтения, в то время как параметр «Базы данных: транзакций/с» — нет.|
